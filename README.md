# tinymce-imageupload
Bower implementation of tinymce-imageupload plugin by [PerfectlyNormal](https://github.com/PerfectlyNormal/tinymce-rails-imageupload)
## Use
```js
tinymce.init({
  plugins: 'uploadimage',
  toolbar: 'uploadimage',
  external_plugins: {
    'uploadimage': 'bower_components/tinymce-imageupload/uploadimage/plugin.js'
  }
})
```
